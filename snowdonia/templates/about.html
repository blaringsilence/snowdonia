About

With Snowdonia's transportation force nearing 1000 vehicles, between trains, trams, taxis, and buses, it's become increasingly important to collect data about the entire transportation system in order to optimize its use, as well as improve it in the future.

This API provides endpoints to collect, as well as provide data relating to the public transportation system.

Send Data

To register a public transport vehicle, send a POST request to:
{{ url_for('register_emission', vehicleID='INSERT_VEHICLE_UUID4')}}
with the following data:

    {
  • 'latitude': float between -90 and 90,
  • 'longitude': float between -180 and 180,
  • 'type': 'taxi', 'bus', 'tram', or 'train',
  • 'heading': int angle between 0 (true North) and 359,
  • 'timestamp': time in the format DAY-MONTH-YEAR HOUR:MIN:SEC, all numbers with leading zeroes (day, month) and full year (e.g. 2016)
    • }
